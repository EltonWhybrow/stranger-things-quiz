About

What is this?

I'm Elton Whybrow, a Frontend Developer from the South-East of the England.

My family and many other people have enjoyed watching Seasons 1-4 of 'Stranger Things' so much, I thought it would be great to test their knowledge on the show to see if they are real superfans or not. Roll on Season 5.

The Tech

The quiz is made in Javascript using the 'Angular' Framework version 14.
Experimented with 'Angular Material' as the (UI) component library as well as custom SCSS.
The source code is available on GitHub.
